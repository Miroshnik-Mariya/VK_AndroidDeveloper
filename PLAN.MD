# План разработки MVP магазина приложений

## Общая информация

**Проект:** Магазин приложений для Android  
**MVP функционал:**
- Список приложений
- Карточка приложения (детальная информация)
- Установка и удаление приложений

**Стек технологий:** Kotlin, Jetpack Compose, Retrofit, Room, Hilt, Coroutines/Flow
